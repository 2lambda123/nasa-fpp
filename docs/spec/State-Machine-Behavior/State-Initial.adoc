=== State Initial

A *state initial* specifies an initial state transition  

==== Syntax

* `[*]` -> 
<<Scoping-of-Names_Qualified-Identifiers,_qual-ident_>>

==== Semantics

. The identifier specifies the initial state transition. 

==== Examples

[source,fpp]
----
state machine MonitorSm {
    [*] -> Black
}

----
